GPIO Studio has basic keyboard shortcuts in-built, for all the macro kings -

 - Clear All - `CTRL + C`

 - Refresh - `CTRL + F`

 - Build Project - `CTRL + B`

 - Run Project (Remote) - `CTRL + R`

 - Load JSON File - `CTRL + L`

 - Save JSON File - `CTRL + S`

 - Help - `CTRL + H`

 - About - `CTRL + A`
 
 - Open Github Repo - `CTRL + G`

 - Quit App - `CTRL + Q`
